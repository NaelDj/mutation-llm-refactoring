# Repo scripts

OS-specific scripts to run tests and PIT mutation testing.
These scripts are the only supported way to execute tests and PIT.

## Windows
 - Run tests: tools\windows\run_tests.cmd
 - Run PIT: tools\windows\run_pit.cmd

## Unix
 - Run tests: ./tools/unix/run_tests.sh
 - Run PIT: ./tools/unix/run_pit.sh
